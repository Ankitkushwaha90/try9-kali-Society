# Linux Basic Commands

## Navigation

- **ls:** List directory contents.
- **cd:** Change the current directory.
- **pwd:** Print the name of the current directory.

## File Operations

- **cp:** Copy files and directories.
- **mv:** Move (rename) files.
- **rm:** Remove files or directories.
- **touch:** Create an empty file.

## File System

- **mkdir:** Create directories.
- **rmdir:** Remove directories.
- **cat:** Concatenate files and print on the standard output.
- **grep:** Print lines matching a pattern.

## Process Management

- **ps:** Report a snapshot of the current processes.
- **kill:** Send signals to processes.
- **top:** Display Linux processes.
- **killall:** Kill processes by name.

## Package Management

- **apt-get:** Advanced Package Tool, for managing packages.
- **dpkg:** Debian package manager.

## System Information

- **uname:** Print system information.
- **uptime:** Show how long the system has been running.
- **free:** Display amount of free and used memory in the system.
- **df:** Display free disk space.
- **hostname:** Print or set the system's host name.

## Networking

- **ifconfig:** Configure network interface parameters.
- **ping:** Send ICMP ECHO_REQUEST to network hosts.
- **netstat:** Print network connections, routing tables, interface statistics, masquerade connections, and multicast memberships.
- **ssh:** Secure Shell, for secure network communication.
- **scp:** Securely copy files between hosts on a network.
- **wget:** Retrieve files from the web.

## User Management

- **useradd:** Create a new user or update default new user information.
- **userdel:** Delete a user account and related files.
- **passwd:** Change user password.
- **su:** Switch user.

## Permissions

- **chmod:** Change file mode bits.
- **chown:** Change file owner and group.
- **chgrp:** Change group ownership.
- **sudo:** Execute a command as another user.

## Text Processing

- **sed:** Stream editor for filtering and transforming text.
- **awk:** Pattern scanning and text processing language.
- **sort:** Sort lines of text files.
- **uniq:** Report or omit repeated lines.

## Miscellaneous

- **date:** Print or set the system date and time.
- **echo:** Display a line of text.
- **tar:** Manipulate archives.
- **gzip:** Compress or expand files.
- **find:** Search for files in a directory hierarchy.
- **history:** Display the command history list.
- **man:** An interface to the system reference manuals.
- **which:** Locate a command.
